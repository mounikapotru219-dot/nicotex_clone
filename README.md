# Nicotex Clone — Smoking Rehabilitation Product

This repository contains a full-stack web application (scaffold) for a smoking rehabilitation product website.

Features
- Frontend: Next.js (App Router)
- Backend: Node.js + Express
- Database: MongoDB (Mongoose)
- Auth: JWT
- Shipping integration: Shiprocket API

Structure
- `frontend/` — Next.js app (App Router)
- `backend/` — Express REST API with Mongoose models

Getting started
1. Copy `.env.example` to `.env` and update variables.
2. Run `npm install` in both `frontend` and `backend`.
3. Start backend: `npm run dev` in `backend`.
4. Start frontend: `npm run dev` in `frontend`.

# Backend (Express + Mongoose)

Run:

1. Copy `../.env.example` into `backend/.env` or set env variables.
2. npm install
3. npm run seed (to create sample admin and products)
4. npm run dev

APIs:
- /api/auth — admin authentication
- /api/products — product CRUD and listing
- /api/orders — place and manage orders (Shiprocket integration)
